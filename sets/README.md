# 📂 Sets – Python Practice Programs

This folder contains Python programs demonstrating the usage of **sets**, one of the most important built-in data structures in Python. Sets are widely used for handling **unique data**, performing **mathematical set operations**, and solving real-world problems such as text comparison and duplicate removal.

---

## 🧠 Concepts Covered

The programs in this folder cover:

* Creating and using sets
* Removing duplicate elements
* Set algebra operations:

  * Union
  * Intersection
  * Difference
  * Symmetric Difference
* Checking for common elements
* Applying sets to text processing problems

---

## 📁 Programs Included

### 1. `remove_duplicates.py`

Removes duplicate elements from a list by converting it into a set.

**Example:**

```
Input:  [1, 2, 2, 3, 4, 4]
Output: [1, 2, 3, 4]
```

---

### 2. `unique_element.py`

Finds elements that appear only once in a list.

**Example:**

```
Input:  [1, 2, 3, 3, 4, 2]
Output: [1, 4]
```

---

### 3. `has_common_elements.py`

Checks whether two lists share at least one common element.

**Concept used:** set
