# Auto Calendar Generator (Python)

## 📌 Overview
The **Auto Calendar Generator** is a Python-based mini project that generates a formatted monthly calendar for a given year and month.  
The program automatically handles leap years and calculates the correct starting day of the month using logical date computation.

This project is built using **core Python concepts** without relying on external libraries.

---

## 🎯 Features
- Accepts year and month as user input
- Automatically detects leap years
- Calculates the correct starting weekday of the month
- Displays a neatly formatted monthly calendar
- Uses modular and reusable functions

---

## 🛠 Concepts Used
- Functions
- Loops
- Conditional statements
- Lists and indexing
- Input validation
- Date logic (leap year calculation)
- String formatting for aligned output
