# Python Mini Projects

This folder contains small Python mini projects created to practice core Python concepts.
Each project is console-based and focuses on logic building, input handling, and clean code structure.

---

## List of Mini Projects

### 1. Anagram Checker
**File:** anagram_checker.py
Checks whether two given strings are anagrams of each other.

**Concepts Used:**
- Strings
- Sorting
- Conditional statements
- Functions

---

### 2. ATM Program (Basic)
**File:** atm_program.py
Simulates a basic ATM system with PIN authentication and withdrawal logic.

**Concepts Used:**
- Loops
- Conditional statements
- User input
- Error handling

---

### 3. ATM Program (Using Functions)
**File:** atm_program_func.py
Improved ATM simulation using functions for better modularity.

**Concepts Used:**
- Functions
- Global & local variables
- Loops
- Input validation

---

### 4. Contact Book (Simple)

### 5. Contact Book (Using Functions)
