# Conditional Statements

## Description
Programs based on **decision-making logic** using `if`, `elif`, and `else`.

---

## Programs Included
- Leap year checker
- Positive / Negative / Zero checker
- Vowel or consonant checker
- Grade calculator

---

## Concepts Practiced
- Conditional branching
- Comparison operators
- User input handling

---

## Learning Outcome
Developed clarity in **logical decision making** and flow control.
