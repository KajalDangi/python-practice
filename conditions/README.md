# 📂 Conditions – Python Practice Programs

This folder contains beginner-to-intermediate Python programs that focus on **conditional statements** such as `if`, `elif`, and `else`.
These programs demonstrate decision-making logic, input validation, and real-world problem solving using conditional branching.

---

## 🧠 Concepts Covered

* `if`, `elif`, `else` statements
* Nested conditions
* Logical operators (`and`, `or`, `not`)
* Input validation using `try-except`
* Boolean-returning functions
* Clean program structure using `main()` and `__name__ == "__main__"`

---

## 📁 Programs Included

### 1. **check_leap_year.py**

Determines whether a given year is a **leap year** based on Gregorian calendar rules.

**Concepts used:**

* Nested conditions
* Logical operators
* Function-based design

---

### 2. **check_positive_negative_zero.py**

Classifies a number as:

* Positive
* Negative
* Zero

**Concepts used:**

* Conditional branching
* Boolean-returning helper functions

---

### 3. **check_vowel_or_consonant.py**

Checks whether an input character is:

* A vowel
* A consonant
* Or invalid input

**Concepts used:**

* String validation
* Character checks
* Input constraints

---

### 4. **grade_calculator.py**

Assigns a grade based on marks entered by the user.

**Example grading scale:**

| Marks  | Grade |
| ------ | ----- |
| 90–100 | O     |
| 80–89  | A     |
| 70–79  | B     |
| 60–69  | C     |
| 33–59  | D     |
| < 33   | Fail  |

**Concepts used:**

* Range checking
* Nested conditional logic
* Input validation

---

## ▶️ How to Run Programs

Make sure Python is installed, then run any file:

```bash
python check_leap_year.py
```
