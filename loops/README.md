# Looping Concepts

## Description
Programs based on iteration using `for` and `while` loops.

---

## Programs Included
- Fibonacci series
- Prime number check
- Table of a number
- Sum of N natural numbers
- Word filtering
- Rotate list

---

## Concepts Practiced
- Loop control
- Nested loops
- Logical iteration

---

## Learning Outcome
Improved problem-solving using loops.
