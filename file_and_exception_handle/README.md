# File & Exception Handling

## Description
Programs demonstrating safe file operations and exception handling.

---

## Programs Included
- Read a file safely
- Write student names to a file
- Append logs
- Copy file content
- Check if file exists
- Error-proof input
- `else` and `finally` blocks

---

## Concepts Practiced
- File modes (`r`, `w`, `a`, `x`)
- `try-except-else-finally`
- Handling `FileNotFoundError`
- Input validation

---

## Learning Outcome
Built confidence in **real-world file-based programs**.
